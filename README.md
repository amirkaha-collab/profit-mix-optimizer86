# Profit Mix Optimizer v3.6

אופטימייזר לתמהיל מסלולי קרנות השתלמות.

## שיפורים בגרסה 3.1

### פיצ׳ר 1 – ייבוא דו"ח מסלקה 📂
- העלאת קובץ Excel של דו"ח מסלקה
- חילוץ אוטומטי של ההחזקות הנוכחיות
- חישוב פרמטרי חשיפה משוקללים
- כפתור "מלא יעדים לפי מצב נוכחי"

### פיצ׳ר 2 – כפתורי מסלול מהיר 🚀
- 5 כפתורים: מניות / אג"ח / חו"ל / ישראל / מט"ח
- מנווט לטאב השוואת מסלולים עם סינון אוטומטי (≥90%)

### פיצ׳ר 3 – נעילת קרן עם משקל קבוע 🔒
- הזנת סכום כולל + סכום לקרן הנעולה
- חישוב אוטומטי של אחוז המשקל
- האופטימייזר שומר על המשקל הנעול

### פיצ׳ר 4 – השוואת מצב קיים vs מוצע 📊
- כל חלופה מציגה טבלת לפני/אחרי (אם פורטפוליו טעון)
- חצים צבעוניים: ▲ שיפור / ▼ הרעה
- badge: שינוי מסלול / ניוד חלקי / ניוד מלא

## התקנה

```bash
pip install -r requirements.txt
streamlit run app.py
```

## משתנה סביבה

```bash
# הגדר סיסמה (ברירת מחדל: 1234)

```

## מקורות נתונים
נתוני הקרנות ושירות נטענים מ-Google Sheets. עדכן את `FUNDS_GSHEET_ID` ו-`SERVICE_GSHEET_ID` בראש הקובץ.

## שיפורים בגרסה 3.1
- עיצוב פרמיום חדש לדף הראשי (header כהה ומקצועי)
- הסרת ה-metrics המיותרים (מסלולי השקעה) מהכותרת
- כפתורי מסלול מהיר עם כותרת ברורה + הסבר על 90%
- הסרת "צעד משקלים" ו"דירוג ראשי" מממשק הגדרות (פרמטרים מיטביים קבועים)
- נקודת מוצא: כל 4 הפרמטרים מסומנים, קשיחות "רך", כיוון "בדיוק"
- הסרת טאב "שקיפות" מהניווט (נשאר בקוד לדיבוג)
- 4 חלופות מומלצות: משוקללת / מדויקת / שארפ / שירות
- טבלת השוואה מהודקת עם מספרים ממורכזים
- גרף Radar מעודכן ל-4 חלופות
